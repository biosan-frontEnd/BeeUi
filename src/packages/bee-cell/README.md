# bee-cell

> 基础组件-cell - 小程序组件

## Install

``` bash
$ npm install @b1/bee-cell -S
```


## API

### Cell

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`prop-name`           | 描述属性的类型，默认值等         |
|`method-name`         | 描述方法的参数，返回值等         |
